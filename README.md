# NexysWEB
NexysWEB is a web browser that does not force you to log in with any account and that makes it use less resources than other browsers. The browser is based off Electron, so it basically is just chromium with less buttons.

## Why NexysWEB?
NexysWEB is a great choice if you want to browse the web without any distractions, and you can (sort of) relive the old days with Wiby! the search engine that I chose because its not google or a mojor platform and it has websites that dont have alot of distractions.
